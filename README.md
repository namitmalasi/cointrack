# Coin Track

CoinTrack is a cryptocurrency tracker application, where you can :

- add and delete the cryptocurrency.
- view the details of every cryptocurrency selected.
- persist the data in localStorage.

# Tech Stack

- React Hooks
- Context Api
- React Router
- Chartjs
- Bootstrap
